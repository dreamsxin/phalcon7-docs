Class Phalcon\Db\Adapter\Pdo\Postgresql¶

extends abstract class Phalcon\Db\Adapter\Pdo

implements Phalcon\Events\EventsAwareInterface, Phalcon\Di\InjectionAwareInterface, Phalcon\Db\AdapterInterface

Source on GitHub

Specific functions for the Postgresql database system

<?php

 $config = array(
  "host" => "192.168.0.11",
  "dbname" => "blog",
  "username" => "postgres",
  "password" => ""
 );

 $connection = new Phalcon\Db\Adapter\Pdo\Postgresql($config);
